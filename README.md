# iOSStudy
iOS 简易教程：知识整理

#### 简介
这个一个简单的 iOS 教程，主要内容是帮助你怎样学习 iOS 教程，帮你找到官方以及其它学习资料。

#### 推荐
* [官方开发者网站](https://developer.apple.com/)，包含各种教程以及知识，如设计、视频、文档等，几乎所有的都能在这里找到。
* [航哥-Swift专区](https://www.hangge.com/)，适合新手的教程。
